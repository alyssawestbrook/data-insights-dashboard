# Data-Insights-Dashboard

beginner friendly application that allows for you to upload datasets to quickly generate summary statistics and charts. Built with Python/Flask, containerized with Docker, and deployed through Google Cloud.

## Core Features

Upload CSV files
Users can upload CSV datasets directly through the application.

Automatic Summary Statistics
The app generates creates statistics using pandas.

Basic Visualization
At least 1 simple chart is automatically created with the datasetâ€™s first numeric column.

Dockerized Application
Fully containerized using a Dockerfile for deployment.

Cloud Deployment
Deployable through Google Cloud Run using Container Registry.
